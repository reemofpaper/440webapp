Code Vs. Zombies is an desktop application that allows the users practice what they know about coding. The user is able to decide what subject they want to test their understanding in. This is done by uploading information files that will generate questions.

This application is intended to be a fun way for children who have never been exposed to coding components to work on their skills and comprehension of different topics.

This application was created as a group project for CS 440 Spring 2019 @ UIC by Daisy Arellano, Hiba Hashim, Ayat Diab, and Reem Hussein.
